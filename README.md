# material_impact_calculator

A web application that estimates the environmental impact of clothing materials using OCR and AI.  
Users can upload a photo of a clothing label or manually select materials to calculate CO₂ emissions and other sustainability metrics.

---

## 🌍 Features

- Upload a clothing label image  
- Automatic OCR extraction (EasyOCR + PaddleOCR)  
- AI-powered material interpretation  
- CO₂ emission estimation  
- Manual material selection option  
- Clean, modern UI with custom branding  
- Fully responsive Streamlit interface
